Welcome! This project is a simple Python password manager developed in my first year studying software development
at the Southern Alberta Institute of technology.

I chose this project becuase I wanted to explore using SQL and encrytption together in a project. 

To initialize the project, you will need python 3.13 or above. Simply download the project and launch the 
main scriupt to get started. 

How it works:
This project was ddveloped using the cryptography module in python. Using PKFD2/Algom2 encryption,
you can securely store passwords locally. Simply choose a Master password which will serve as 
the key for all your other passwords and get started storign your passwords. 

Skills learned:
-Object oriented design
-Singleton design pattern
-Salting and hashing
-SQLite3 datbase connection and commands
